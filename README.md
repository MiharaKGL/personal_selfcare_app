# personal_selfcare_app

🌸 SelfCare Android App

A simple and beginner-friendly Self Care Android Application built using Android Studio. This app was created as a mini project to help users improve their daily well-being through self-care activities and tracking features.

✨ Features


  👤 User Registration Screen
  
  💬 Positive Quote Generator
  
  😊 Mood Tracker
  
  🧘 Physical Well-Being Tracker
  
  📓 Daily Reflection Journal

  ⚖️ BMI Calculator
  
  🎨 Cute and calming UI design
  
  📱 Screens Included


The application contains:

  Registration & Login UI
  Home Dashboard
  Physical Well-Being Tracker
  Mood Tracker
  Daily Reflection Journal
  BMI Calculator

  
🛠️ Built With
  Java / Kotlin (depending on your project)
  Android Studio
  XML Layouts
  Android SDK

  
⚠️ Limitations

This is a mini project created for learning purposes.

Current limitations:

  No database integration
  
  User data is not permanently stored
  
  No cloud connectivity
  
  Limited authentication features
  

Future improvements may include:

  Firebase integration
  
  User authentication
  
  Data persistence
  
  Notifications & reminders
  
  Dark mode support
